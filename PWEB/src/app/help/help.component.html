Comment utiliser PManager?

Création de compte/Connexion

1. Dans profil, connectez-vous à l'aide de votre compte Google.

2. Un compte Google est nécessaire pour l'utilisation de notre site.

Création de projet

1. Connectez-vous à PManager à l'aide de votre compte Google.

2. Dans "Partage de fichiers", cliquez sur Nouveau Projet.

3. Entrez le nom de votre projet ainsi que les (adresses) des membres.

4. Votre projet apparaît alors dans la section "Vos Projets *".

Upload un fichier ou un dossier

1. Connectez-vous à PManager à l'aide de votre compte Google.

2. Dans "Partage de fichiers", cliquez sur "Tous mes fichiers" pour apercevoir l'ensemble de vos fichiers mis en ligne.

3. Choisissez un fichier localement, puis appuyez sur "Upload" pour les mettre en ligne.

Nous contacter

1. Dans la page Contact, laissez-nous votre adresse e-mail, le titre, puis le contenu de votre demande.
